NeuroCine Director v4

1. Upload this project to GitHub/Vercel.
2. Add environment variables:
   OPENAI_API_KEY
   OPENAI_MODEL=gpt-5.4
   ANTHROPIC_API_KEY
   ANTHROPIC_MODEL=claude-sonnet-4.5
3. Run npm install && npm run build.

No Groq. No upscale. Main stack: GPT-5.4 + Claude Sonnet 4.5 fallback.
