# NeuroCine Build Fix Patch

## Исправлено

1. Удалён конфликтующий файл `app/api/admin-credits/page.js`.
   - Он лежал внутри `app/api/admin-credits` рядом с `route.js`.
   - В Next.js App Router нельзя держать UI `page.js` и API `route.js` в одном route-сегменте.
   - Из-за этого Render/Turbopack падал на build.

2. Админка остаётся по правильному пути:
   - `/admin/login`
   - `/admin`

3. API начисления остаётся по правильному пути:
   - `/api/admin-credits`

4. Валюта приведена к формату `⭐`.
   - UI показывает `⭐ 99`, `⭐ 1` без слова “Синема”.

5. Исправлен стартовый баланс:
   - новый пользователь получает `25 ⭐`;
   - дневной минимум — `10 ⭐`, без обнуления большого баланса.

## Что удалить вручную, если заменяешь файлы поверх старого проекта

Удалить файл:

```text
app/api/admin-credits/page.js
```
